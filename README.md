# MayaFit – محاسبه‌کننده آنلاین BMI

![Logo](IMG_1198.png)

MayaFit یک ابزار ساده و کاربردی برای محاسبه شاخص توده بدنی (BMI) است.  
با وارد کردن وزن و قد، این برنامه به سرعت BMI شما را محاسبه کرده و دسته‌بندی وزن شما (کم‌وزن، طبیعی، اضافه‌وزن یا چاق) را نمایش می‌دهد.  

این ابزار به شما کمک می‌کند وضعیت سلامت خود را بهتر درک کنید و انتخاب‌های بهتری برای سبک زندگی سالم داشته باشید.  
MayaFit کاملاً آنلاین است و نیازی به نصب ندارد، بنابراین می‌توانید از هر دستگاهی به راحتی از آن استفاده کنید.

## استفاده
1. قد خود را به متر وارد کنید.
2. وزن خود را به کیلوگرم وارد کنید.
3. روی دکمه "محاسبه" کلیک کنید تا BMI و دسته‌بندی آن نمایش داده شود.
